## Web tools list ##
- **Ad and cookie blocker** — blocks ad/tracker domains and strips cookie banners.
- **Download media** — lists direct `<video>`/`<audio>` sources on the page with download links.
- **NetStatBuddy** — estimates your connection stats (see disclaimer below).
- **QR this page** — shows a QR code for the current URL.
- **View page metadata** — dumps title, canonical, description, Open Graph / Twitter tags, and link/image/word counts.
- **Picture in picture** — pops the biggest video on the page into a floating picture-in-picture window (run again to exit).
- **Reader mode** — strips the page down to just the article text on a clean page, with A-/A+ font buttons.
- **Edit any page** — toggles `designMode` so you can click into the page and type/delete anything (great for screenshots). Nothing is saved.
- **Dark mode anywhere** — inverts the page for a dark theme while keeping images/videos looking normal. Run again to undo.
- **Color picker** — eyedropper to sample any pixel's color; shows HEX + RGB with a copy button (Chrome/Edge only).
- **Storage viewer** — inspect this site's cookies, localStorage and sessionStorage, with a one-click "clear all" for the site.

## Disclaimer for NetStatBuddy ##
NetStatBuddy will *estimate* the internet stats when the page you run it on is strict-CSP. What this means is that NetStatBuddy can't fetch the internet stats from a internet stat checker (such as fast.com) if a page restricts it. 
### How to fix it? ###
Run the bookmarklet on a blank page or new tab. You will now get the more accurate stats :)
